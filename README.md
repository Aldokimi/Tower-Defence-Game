# Alpha Tech
Evolve your gaming experience

## Quickstart

### Prerequisites

- JDK 8+<br>
- Gradle 7.4.2

### Build
```bash
gradle desktop:build 
```
or 
```bash
gradle build 
```
### Run the game
```bash
gradle desktop:run 
```
or 
```bash
gradle run 
```

### Run unit tests
```bash
gradle test
```
or
```bash
gradle check
```
or 
```bash
gradle :core:test
```
